Easy to use WordPress slider plugin. Based on boostrap slider.

== Installation ==

This section describes how to install the plugin and get it working.



1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Plugin Name screen to configure the plugin

*Features

* Implementation of bootstrap carousel for wordpress
* Create Responsive, slideshows in seconds
* Simple, easy to use interface - perfect for individual users, developers & clients!
* Support for Image slides (supports caption, link, title text, alt text)
* Full width slideshow support
* Built in  Shortcode
* Configuration options - transition  speed per slideshow
* Fast - only the minimum JavaScript/CSS is included on your page
* Free basic support (covering installation issues and theme/plugin conflicts)
